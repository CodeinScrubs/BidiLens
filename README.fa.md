# BidiLens

BidiLens یک ابزار متن‌باز برای نمایش درست متن‌های ترکیبی فارسی، عربی، عبری، اردو و انگلیسی در رابط‌های هوش مصنوعی است.

سیاست پیش‌فرض `content-majority` است: جهت پاراگراف بر اساس محتوای طبیعی غالب تعیین می‌شود، نه اولین کلمه. بنابراین جملهٔ زیر با وجود شروع انگلیسی، RTL تشخیص داده می‌شود:

```text
React یک کتابخانه جاوااسکریپت بسیار محبوب است.
```

رشتهٔ اصلی هرگز برعکس یا با نویسهٔ کنترلی مخفی تغییر نمی‌کند. برای نمایش، از `dir`، `bdi` و جداسازی ساختاری استفاده می‌شود تا کپی، جست‌وجو، دسترس‌پذیری و ترتیب منطقی متن سالم بماند.

```bash
npm install
npm run check
```

جزئیات معماری و محدودیت‌ها در `docs/ARCHITECTURE.md` و `docs/ROADMAP.md` آمده است.
