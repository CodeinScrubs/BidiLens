# Application notes

This file is an evidence checklist for future open-source, grant, or
sponsorship applications. It is not an application claim.

## Evidence available

- source, tests, corpus, Unicode generator, visual suite, examples, and release
  verifier are committed in one repository;
- the motivating user case is fixture #001 and runs through every shipped
  adapter's test surface;
- comparisons with sibling attempts are documented with reproducible local
  file counts;
- unsupported platforms and external prerequisites are explicitly listed.
- the canonical source and maintainer identity are public at
  `CodeinScrubs/BidiLens` with reproducible CI configured.

## Evidence to obtain before applying

- native-speaker-reviewed corpus samples;
- at least one accepted downstream issue or merged integration;
- users or maintainers willing to describe the bug and result;
- public CI history and provenance-backed package artifacts;
- accessibility and security review appropriate to the claimed audience;
- download/adoption metrics obtained from real services.

Do not estimate or invent any missing number. A smaller verified application is
stronger than broad claims that cannot be reproduced.
