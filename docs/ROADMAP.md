# Roadmap

## v1.0 readiness

- Core direction detector and paragraph analysis
- Stable streaming state machine
- Unicode control audit and sanitize functions
- remark and rehype plugins
- DOM annotator and observer
- React primitives
- CLI and multilingual corpus
- Unit, integration, and benchmark coverage

## v1.1

- Browser screenshot regression suite with Playwright (flagship fixture complete)
- React Markdown integration package
- Vue and Svelte adapters (initial adapters complete; broader framework corpus remains)
- Editor decorations for Monaco and CodeMirror
- Corpus imports from public Unicode bidi test data

## v1.2

- Native Android Compose and SwiftUI reference adapters
- Terminal capability matrix and ANSI-safe isolation utilities
- PDF export guidance and reference implementation
- Accessibility testing against major screen readers

## v2

- Language-aware segmentation policies
- Confusable and mixed-script security module
- Collaborative conformance dashboard
- Browser extension and IDE extension
